HF Flow is a web application that allows users to create and share Huggingface model flows.

## Getting Started

### Setup

Link vercel project
    
```bash
pnpm dlx vercel link
pnpm dlx vercel env pull
```

### Run the development server

Run the development server:

```bash
pnpm dev
```
